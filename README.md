# Portfolio Website

A static HTML/CSS/JavaScript portfolio website.

## Project Structure

- `index.html` - Main homepage
- `styles.css` - Global styles
- `script.js` - JavaScript functionality
- `jdv/`, `missouri/`, `popmore/`, `potential/`, `suisse/` - Individual project pages
- `media/` - Static assets (images, videos, favicons)

## Features

- ✅ Pure static HTML/CSS/JavaScript
- ✅ Optimized images with native HTML attributes
- ✅ Responsive design
- ✅ Smooth horizontal scrolling portfolio layout

## Usage

Simply open `index.html` in a web browser or serve it with any static file server.
